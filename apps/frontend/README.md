# Frontend (Portfolio Admin)

React + Vite frontend for the Portfolio Admin portal.

Authentication is provided via AWS Amplify UI (Cognito).

## Configuration

Create an `.env.local` file (not committed) with:

- `VITE_USER_POOL_ID`
- `VITE_USER_POOL_CLIENT_ID`

## Development

npm install
npm run dev

## Build / Preview

npm run build
npm run preview
